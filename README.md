# Hướng dẫn triển khai dự án

## Phía Client

1. Cài đặt các gói phụ thuộc:

   ```bash
   npm i
   ```

2. Build ứng dụng:
   ```bash
   npm run build
   ```
3. Chạy ứng dụng với HTTP:
   ```bash
   npm run start-http
   ```

## Phía Backend

1. Cài đặt các gói phụ thuộc:

   ```bash
   npm i
   ```

2. Chạy server:
   ```bash
   node src/server.js
   ```
3. Chạy tệp push.js để gửi thông báo Push:
   ```bash
   node src/push.js
   ```
